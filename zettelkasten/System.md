#public 
A system is defined as a set of components, that interact with each other, to achieve a specific goal or function. 

### Key Characteristics
- Structure
- Behaviour
- Boundaries
- Inputs
- Outputs

### Types of systems
- Software systems
- Hardware systems
- Network systems
- Mechanical systems
- Ecosystems
- Abstract systems
- Combinations of all these.
