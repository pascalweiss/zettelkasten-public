#public
A tool for direct TypeScript execution. The code must not be compiled to Javascript, anymore. This also makes TypeScript configurations obsolete. 


### Example usages
```
# installation
npm install -g tsx

# simple execution
tsx script.ts

# hot-reload execution
tsx watch server.ts
```
